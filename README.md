# ClassRoomOptimizationSystem
The scope of this project will provide direction and help one stay on target with regards to the areas covered by this project and in the other hand suggests additional work that might likely be done on the application in the future. This project has two major aspect, the ADMIN and USER side. The Admin using the automatic functions provided by the system to assign Lecture DAY, TIME and VENUE of best capacity to courses to form a standard school timetable. The User on the other hand has access to the system only to view the timetable which has been scheduled by the admin.  Note that the Admin only gives instruction to the system who goes ahead to assign Day, Time and Best Venue to courses automatically. The case study used in this project is University Of Nigeria Nsukka, Computer Science department.
